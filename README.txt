超级评教王 (Super Evaluation King)
（名字致敬传奇的超级乐跑王）

这是一个基于 Python、Tkinter 和 Playwright 的自动化脚本，用于自动完成东北大学（NEU）的学生评教任务。
本工具通过模拟浏览器操作，自动登录并为你所有的课程提交评教。

重要声明
  * 仅供学习交流：本项目仅供个人学习 Python GUI 编程和 Playwright 浏览器自动化技术使用，请勿用于任何非法或违规用途。
  * 本地运行：你的学号和密码仅在你的电脑本地运行，脚本不会将其发送到除东北大学官方登录接口外的任何地方。
  * 使用风险：教务系统的网页前端随时可能更新，这会导致本脚本的自动化流程（选择器）失效。
  * 后果自负：请自行承担使用本脚本可能带来的一切风险，作者不对任何潜在的学期、学业或账户问题负责。

功能特性
  * 自动登录：自动打开浏览器并登录东北大学一网通办。
  * 自动导航：自动跳转到“教学质量监控与评价平台”的“学生阶段评价”页面。
  * 智能评教：
      * 自动检测所有需要“评价”或可以“修改”的课程。
      * 为每一门课程的所有评价项自动选择 “6” （最高分）。
      * 自动提交评教。
  * 即时反馈：评教完成后会弹出消息框提示，并保持浏览器窗口开启10分钟，以便你检查评教结果。

安装与使用
1.  准备环境
首先，克隆或下载本仓库到你的本地。
然后进入项目目录
2.  创建虚拟环境
强烈建议使用 Python 虚拟环境 (venv) 来管理项目依赖。
(例如: python -m venv .venv)
然后激活它：
Windows (CMD/PowerShell):
..venv\\Scripts\\activate
macOS / Linux:
source .venv/bin/activate
3.  安装依赖
本项目依赖 playwright。
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
4.  安装浏览器（关键步骤！）
Playwright 需要专用的浏览器驱动才能工作。运行以下命令来下载它们：
playwright install
5.  运行程序
一切准备就绪！运行main.py
在弹出的“超级评教王”窗口中，输入你的学号和一网通办密码，然后点击“开始评教”即可。
